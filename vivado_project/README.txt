=== vivado_project ===

This directory should contain the ready-to-submit Vivado project after running "reset_project". It includes the .bit bitstream and the cleaned project folder (Project/) without unnecessary generated files.

本目录应放置执行“reset_project”命令后干净的Vivado工程，包含.bit比特流文件以及清理过的Project/文件夹，不含多余的生成文件。
